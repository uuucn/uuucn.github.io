教程十分钟，少读十年书

开始使用前您需已确认以下内容

  • 我的设备同步了北京时间并且误差小于30s.
  • 我已经确认我购买的套餐没有过期并且流量没有用完而且账户没有被封禁
  • 我已确认我会复制粘贴等电脑操作
  • 我知道如何正确仔细的阅读教程并且没有漏掉一个任何的文字
  • 我知道如何正确提问并且会使用Google/Baidu等搜索引擎，并且我已完成中国大陆的九年义务教育
  • 我的网络不是校园网或者公司内网

如果你正在使用电脑

  • 如果你是在浏览器上我已卸载所有代理插件
  • 我没有同时打开多个代理软件或者加速器或者游戏加速器
  • 我的电脑中没有360，或者我确保360软件不会干扰我的加速工具

iOS 教程

Shadowrocket（小火箭）

简介

Shadowrocket是ios平台上较早出现的支持SSR协议的客户端，坊间称之为“小火箭”。

软件的界面直观，上手难度低。

系统环境：

iPhone XS / iPhone os15.6.1

Shadowrocket 2.2.21（1808）

下载安装

小火箭在中国大陆已被下架，请使用非国区账号在AppStore下载 价格：2.99＄。

添加订阅

  1. 扫码导入

点击小火箭左上角的扫码按钮，自动识别并添加。

订阅链接

  1. 打开 Shadowrocket 点击右上角 + 按钮

在新的页面中，选择 「类型」为 Subscribe

在「URL」一栏粘贴自己的 订阅链接 ，最大值 一栏不填写 备注 一栏随意填写。

点击右上角的完成 Shadowrocket 会从订阅服务器获取订阅信息并添加节点。

启动服务

在上面的步骤中，已完成基本设置。

延迟测试

点击 连通性测试 若干秒之后节点右侧会有数值显示。

此延迟与网速并无关系 延迟测试方法 选择 CONNECT 真实链接

选择节点

建议选择一个延迟较低的节点后，打开顶部的开关。

如果是第一次使用，则会提示需要添加 VPN 连接。

此时点击 Allow 后按照系统提示操作，验证密码 或 Touch ID

之后会自动返回到 Shadowrocket 界面并启动服务。

此时已完成最基本的设置，已经可以开始使用。

Android 教程

Clash For Android

简介

Clash 是一个使用 Go 语言编写，基于规则的跨平台代理软件核心程序。 Clash for Android 是安卓系统上的一款 Clash 客户端。 Github 项目地址：https://github.com/Kr328/ClashForAndroid

快速上手

Clash For Android 支持两种导入配置的方式：

1.

URL（订阅链接）导入

2.

本地配置文件导入

订阅导入

登录机场官网，进入首页，下拉页面，找到便捷导入，复制clash订阅地址。

复制号 Clash 订阅链接后，打开 Clash For Android 应用程序。

点击配置按钮，在新弹出的窗口中点击新配置。

已复制 Clash 订阅链接的用户请点击 URL ，并粘贴您的 Clash 订阅链接。然后点击选中自己的配置。

—–无图—–

返回首页。点击开关，即可打开代理。首次使用会提示是否允许建立VPN连接，点击允许即可。

—–无图—–

开启代理之后，可点击代理选项卡，进入分流策略组面板，在这里可以切换节点。选择你想要的节点即可进行切换使用。

下滑进行延迟测试或点击顶部右上角按钮进行，测试结果将会现实在节点名称的右侧。

—–无图—–

到这里，基本上基础的使用教程就好了，目前已经可以进行科学上网了。

本地文件导入

点击配置—新配置—从文件导入，然后从本地文件夹选择自己要导入的配置文档。

Windows 教程

Clash For Windows使用教程

该软件需要一定的技术才能正常使用,若无本领请勿死磕

简介

Clash 是一个使用go语言编写，基于规则的跨平台代理软件核心程序。

Clash for Windows 是运行在 Windows 上的一图形化分支。通过clash API 来配置和控

制 clash 核心程序，便于用户的可视化操作跟使用~

目前已支持 SS / SSR / VMESS / Trojan~

整理教程时使用的系统环境：

Windows 10 21H2 （OS内部版本 19044.2006）

Clash for Windows 0.20.6

下载及安装

直接访问 GitHub 地址进行下载：点击跳转

GitHub 下载的是未经过汉化的英文版，个人建议尽量使用原版。

快速上手

  1. 在机场官网首页复制 Clash 订阅链接。

  1. 打开 clash for Windows 应用程序，在左侧的标签页中选择 Profiles，在顶部粘贴刚

才复制好的 clash 订阅链接，然后点击 Download 按钮。

  1. clash for Windows 会自动拉取配置文件进行下载，如果一切顺利，你就可以看到绿色的 <Success!>，并且在下方会有一个新增的配置文件~

  1. 点击新增的配置文件来切换到该配置，然后点击< proxies> 标签页来切换接入点，将顶部

    的出战模式选择为 < Rule >。在该模式下你的网络访问请求都将通过 Clash for Windows 进行

    分流处理。

    在< proxies> 策略组中选择你想要使用的接入点。 proxy 策略组是用于访问网络的默认策略

    在 Proxy ，选择你想调用的策略组。

    在不进行其他修改的情况下，所有网络的访问都通过 proxy 策略组中选择的接入点进行。

    1. 返回到< General> 标签页下，将 < System Proxy > 的开关更改为绿色或者 < on > 的状态

    即可开始使用。此外，如果你想让 clash for Windows 开机自启的话，需要将 < Start with windows > 也更改为绿色或者 < on > 。

从本地导入YAML文件

在无法直接从软件里面更新自己的clash订阅时，可以复制你的clash订阅链接，直接粘贴到浏览器打开，然后复制全部文本，在桌面新建一个文本文档（TXT格式），然后把复制的文本粘贴到文档里面，保存后将文件的扩展名称更改为YMAL。然后可直接拖拽到软件的 proflies界面。 这应该算是一种曲线救国的方法吧，毕竟使用该方法导入的配置是无法进行在线更新的，需要更新的时候，需要重新来一遍~

到这里，基本上基础的使用教程就好了，目前已经可以进行科学上网了。
