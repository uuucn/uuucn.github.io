美区 Apple ID 详细注册教程

发表于2022-05-20

绝大多数时候，国区的 Apple ID 基本上可以满足大多数国内用户的所有需求。

不过如果你想要下载一些外区 Apple Store 才上架的 ，那就需要使用外区的 Apple ID 登录外区的 App Store。

注册外区 Apple ID 本身是不需要钱的 ，出于安全因素考虑，不建议使用从淘宝等第三方渠道购买的未知来源的外区 Apple ID。

开始注册

开始注册前，建议打开科学上网工具并开启全局模式。

  1. 在自己的电脑或手机上访问 https://appleid.apple.com 点击右上角的 创建您的Apple ID 。

  2. 在打开的 创建您的 Apple ID 页面，按照页面指示完善个人信息。

    使用英文名注册，国家和地区选择 美国，电话号码可以填国内手机号，

    建议使用 Gmail 邮箱或 Outlook 邮箱注册；验证方式选择 短信。

  3. 输入验证代码后，点击 继续 按钮。

验证通过后，页面将自动登录刚刚申请好的 Apple ID，点击左侧边栏个人信息，可以看到此 Apple ID 的国家或地区显示为美国。

由于我们新注册的 Apple ID 尚未在 iTunes 商店使用过，我们需要在接下来的步骤完善这个 Apple ID 的地址信息。

完成创建 ID

  1. 添加支付方式

  1. 按照页面指示完善个人信息，付款方式选择 “None“ 账单寄送地址填入美国地址信息。

    美国地址信息使用 美国地址生成器 建议使用免税州地址信息。

    随后点击 “Update“ 就可以开始在美区的 App Store 下载安装 app 了。

    登录上 App Store 后建议先下几个免费应用。
