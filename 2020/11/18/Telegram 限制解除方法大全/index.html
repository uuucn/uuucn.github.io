Telegram 限制解除方法大全

发表于2020-11-18

频道/群限制

Telegram 如果你进入某个群或频道遇到如下提示：

This channel can’t displayed because it was used to spread pornographic content.

原因：

有人在群里发了色情内容，被人举报后，被 Telegram 官方限制了；

Telegram 官方只会处理举报的，不会主动审查，如果没人举报就不会被限制.

有两种情况：

只有 iOS & Mac 上 App Store里下载的客户端会有此限制，这是因为 Apple 的限制，上架App Store 的内容限制；其它客户端无此限制，可以正常进入群，可以用下面方法解除限制；
所有客户端和网页版都不能进入此群和正常显示此群消息，那说明是永封，无解。
解除限制的操作方法：

四个方式开启：
方式a：打开 Desktop 电脑桌面客户端，左上角三短线: 👉 设置 👉 隐私安全 👉 开启”显示含有敏感内容的媒体”

方式b：打开 MacOS 电脑桌面客户端：👉 设置 👉 隐私安全 👉 开启”显示含有敏感内容的媒体”

方式c：网页版新版：手机或电脑浏览器访问官网网页版地址：https://web.telegram.org

输入手机号码登录(验证码是优先发送到之前已登录的 Telegram 设备/客户端/App上)

主界面 👉 左上角三短线 👉 Settings 👉 点击Privacy and Security👉 滑动到底部 👉 打开Disable filtering.

注意： 必须用浏览器打开，验证码是优先发送到之前已登录的 Telegram 设备/客户端/App上.如果你在 Telegram 直接打开链接，就无法看到 Telegram 上的验证码。

方式d：网页版旧版：手机或电脑浏览器访问官网网页版地址：Telegram Web

输入手机号码登录(验证码是优先发送到之前已登录的 Telegram 设备/客户端/App上)

主界面 👉 左上角/右上角三短线 👉 Settings 👉 打开”Show Sensitive Content”

注意： 必须用浏览器打开，验证码是优先发送到之前已登录的 Telegram 设备/客户端/App上.如果你在 Telegram 直接打开链接，就无法看到 Telegram 上的验证码。

重启 Telegram iOS 客户端即可，或者等几分钟再重启下，或者重新登录.
整体的过程就是桌面端或者网页版开启显示，手机端就可以了，不需要在手机端设置中操作.
要求：

Telegram iOS 客户端版本 ≥ 5.13
备注：Windows，MacOS，Linux 系统的电脑都可以使用 Telegram Desktop 版本客户端

+86私聊限制

Telegram 给某人发送私聊信息时遇到如下提示：

Sorry,you can only send messages to mutual contacts at the momet.

中文：

抱歉，目前您被限制无法主动发起私聊信息。

抱歉，您现在只能给双向联系人发送消息。

原因：

之前币圈利用 Telegram 的便利，大量发广告和拉人进群。

Telegram 限制了+86 大陆手机号绑定的账号的私聊。

限制情况：

+86 的账号不能给非 +86 的账号主动发起私聊

+86 的账号能给 +86 的账号主动发起私聊的；

如果是别人给你发起的私聊，你是可以回复交流的；

刚刚注册的 +86 账号也不能发起私聊的，过段时间就OK了。

还有这个 Bot 是不会回复你 +86 的这个限制信息的，不能仅从 Bot 回复看你是否是 +86 限制

解决办法：

换绑 Google Voice 可永久解决。
向官方申诉，但似乎隔段时间又得重新申诉。
*向官方 @SpamBot 发送消息申请(跟着提示操作就行.)，忽略回复的 ‘You’re free as abird!’ 这个说的是你没有被全局限制，和 +86 这个限制没关系，不能仅从Bot回复看你是否是 +86 限制

*第一步点击选项：But I can’t message non-contacts!

*第二步点击选项：No I’ll never do any of this!

*第三步输入内容：accodent ,发送

*大约30分钟左右即可解除限制，有可能更长

*30分钟左右还未解除的就继续等等或再次提交

*Bot 不会回复你结果，自己找人私聊测试

*如果 Bot 回复和教程的不一样，仔细看 ↓

你的账号被 Telegram 官方限制了，不能在公开群和(或)主动私聊发送信息

一般是在群里和私聊发消息被人多次举报就有可能被限制

私聊举报和群里管理员的举报+ ban 的权重比较大
但无法查到被谁举报的，也无法查到是因为哪条消息举报的

这个限制和哪个号码注册没有关系，所有号码注册都有可能被如此限制，和 86 限制无关.

你的 Bot 回复的消息里有说明解封时间，到时见会自动解封，不能提前解封.

不过是 UTC 时间，比北京时间晚 8 小时，北京时间=UTC时间+8小时。

如果 Bot 回复中没有时间，那说明不会自动解封，这种情况的具体原因不清楚，只有问问官方了。

加群一直转圈

Telegram 点击群组和频道链接一直 转圈 加载

有两个可能：

短时间内打开群组和频道链接和加群太多，被 Telegram 官方限制了，等几个小时或24小时后再操作。

在一天之内，若用户访问超过200个群组或者频道的链接 点击打开就算访问，不需要加入，那么该用户会被限制数个小时，在限制时间内是无法通过链接打开任何群组或者频道，具体表现为点击链接一直转圈而无法访问。

还有可能你的网络/代理网络慢
