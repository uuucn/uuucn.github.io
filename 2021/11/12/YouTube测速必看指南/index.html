YouTube测速必看指南

发表于2021-11-12

如何查看详情统计信息

网页端

在浏览器中打开 YouTube 视频 👉 鼠标移动至视频画面区域 👉右键 👉 选择 详细统计信息


移动端

在手机(安卓 / iOS) 、平板(安卓 / iPadOS) 等移动设备

首先 需要在 App 里打开 详细统计信息 功能

👉 点击右上角的头像 👉 设置 👉 常规 👉 打开 详细统计信息

然后打开视频 点击右上角的设置按钮

各项参数是什么意思

  • Video ID / sCPN = 每个视频的唯一 ID 号
  • Viewport / Frames = 当前播放视窗的大小 /dropped 视频丢帧情况
  • Current / Optimal Res = 当前 / 视频推荐分辨率大小
  • Volume / Normalized = 当前播放音量大小(content loudness) 和 YouTube 推荐标准音量的差异值
  • Codecs = 编码器
  • Connection Speed = 链接速率(当前客户端到达此视频服务器的实时链接速率)
  • Network Activity = 实时的网络活动
  • Buffer Health = 缓冲时间(断网以后视频可以继续播放的时间)
  • Mystery Text = 神秘代码
  • S = 播放状态 (4 暂停、8 播放、14 加载视频中等等、19快进或快退)
  • T = 视频的时间轨道线
  • B = 缓冲区间和缓冲区大小(缓冲区的第二数字 - T (时间轨道线) = 缓冲的时间)
